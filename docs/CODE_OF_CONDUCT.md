# Code of Conduct

Everyone interacting with this repository is expected to follow this Code of Conduct.

## Standards

- Be respectful and professional.
- Assume positive intent but be clear and direct.
- No harassment, hate speech, or discriminatory behavior.

## Scope

This applies to all project spaces, including issues, PRs, discussions, and any
other communication channels related to this project.

## Enforcement

- Violations may result in blocked accounts or removal of contributions.
- Serious or repeated violations will not be tolerated.
